# 📅 Date Converter 

<div align="center">
  <p>A simple, efficient, and robust tool designed to seamlessly convert dates between different calendar systems (e.g., Gregorian to Hijri, or vice-versa).</p>
</div>

---

### ✨ Features
- 🔄 **Bi-directional Conversion:** Easily convert from Gregorian to Hijri and vice-versa.
- ⚡ **Lightweight & Fast:** Minimal dependencies ensure optimized runtime execution.
- 🛠️ **Developer Friendly:** Clean code architecture, making it easy to embed into larger platforms, APIs, or mobile apps.
- 🌍 **Localization Ready:** Built with formatting parameters suitable for region-specific adjustments.
